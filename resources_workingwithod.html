Using the opendata.scot data

Metadata for datasets listed on opendata.scot can be accessed in JSON format Daily JSON file dumps of our Analytics data can be accessed at https://github.com/OpenDataScotland/opendata.scot_analytics

To learn more about how the data is processed see the Tools & Tech section of our documentation
